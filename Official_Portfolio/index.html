Video production

Inventory System

A web application built with a simple goal in mind. To allow students to borrow Video production equipment, in the efforts of replacing the slow paper version of its former self.

Technology

Video production Inventory System uses Bootstrap3 for a fast layout, with JavaScript to enhance student’s interaction and responsiveness. On the backend, it uses PHP and Ajax for dynamic mail handling. The client “The Video Production Teacher” required the ability to see what the student had borrowed and automatically deduct the number of pre-existing equipment into an excel worksheet. Microsoft VBA was used to interact with a firebase API to grab data that got pre-set by the web application. a json API was used to converter json format data to VBA.
