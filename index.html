Video production

Inventory System

A simple web application built with a simple goal in mind. To allow students borrow Video production equipment, in the efforts of replacing the slow paper version of its former self.

Technology

Video production Inventory System uses Bootstrap3 for a fast layout, with JavaScript to enhance student’s interaction and useability. On the backend it uses PHP for mail handling as well as ajax. The client (“The video production teacher”) required the ability to see what the student had borrowed and automatically deduct the number of pre-existing equipment into an excel worksheet, so it was a must to use Microsoft VBA to grab data from firebase and a json API acting as a bridge between the two programs.
